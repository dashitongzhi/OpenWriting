# 曼波节奏音轨素材说明

`manbo-bed.mp3` 是为 OpenWriting 宣传片编排的原创节奏床，不包含从抖音或其他短视频平台提取的音频。

打击乐单击采样来自 [Versilian Community Sample Library (VCSL)](https://github.com/sgossner/VCSL)，包括 conga、quinto、bongo、cowbell、claves 与 shaker。VCSL 以 [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/) 发布，可用于商业及非商业媒体，无需署名。本工程仍保留来源说明，方便审计和复现。

节奏、旋律、低音编排与最终混音由本项目的 `scripts/generate-manbo-track.mjs` 生成。
